# OpenMeteo Weather App

A modern, responsive Flutter application providing real-time weather information using the Open-Meteo API.

## Features

- **Real-time Weather**: Get current weather data based on your GPS location.
- **Multi-language Support**: Instant switching between 8 languages (English, Indonesian, Chinese, Japanese, Korean, Arabic, Russian, Hindi).
- **Responsive UI**: Premium design with smooth animations and full scrollability.
- **Easy Settings**: Quick access to donation links, credits, and help.
